# Welcome to my SpringBoot CXF WS-CRUD!
## Technologies used:
* Java openJdk-17
* IntelliJ CE
* SOAP UI
* Maven 3.9.6
* Spring Web 3.1.5
* Spring Data JPA 3.1.5
* CXF JAX-WS 4.0.3
* Lombok 1.18.24
* H2-Database 2.2.220

# How to run it?
`mvn clean spring-boot:run`
