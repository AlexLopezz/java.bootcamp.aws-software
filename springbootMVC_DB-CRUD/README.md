# Welcome to my SpringBoot MVC DB CRUD!
## Technologies used:
* Java openJdk-17
* IntelliJ CE
* Dbeaver CE
* Maven 3.9.6
* Spring Boot 3.1.5
* Spring Boot Data JPA
* Spring Boot Validation 3.1.5
* Docker
  * mariadb:11.2
* Lombok

# How to run it? 
### Docker - mariadb
* Must create a new container from mariadb:11.2 w this command:
``` docker run -d -p 3306:3306 --rm --name mariadb mariadb:11.2 ```
