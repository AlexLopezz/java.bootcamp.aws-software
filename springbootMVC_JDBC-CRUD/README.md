# Welcome to my Spring Boot MVC JDBC CRUD!
### Technologies used: 
* Java OpenJDK-17
* IntelliJ CE
* Dbeaver CE
* Lombok 1.18.24
* Spring Boot 3.1.5
* Spring Validation 3.1.2
* Spring Boot Data JPA
* Thymeleaf 3.1.5
* Docker
  * mariadb:11.2
 
### How to run it?
## Docker
  * Open a new terminal and exec this command: ``` docker run -d --rm --name mariadb -e MYSQL_ROOT_PASSWORD=root -p 3306:3306 mariadb ```
